About this project

Because we don't need waste time looking for good content.

Whenever you are new to a certain area of ​​knowledge, you need to learn fast everything about that. It is not always possible to have someone by your side telling you the easy way. What happens is that you end up wasting precious time looking for good sources of information like blogs, books and other important subjects to study. The idea of ​​this project is help you bringing some content, tools and resources to you don't waste your time.

If you want to contribute with the project, please, submit a Pull Request on the GitHub or just open an issue. Easy peasy.

My name is Diego Eis. I'm Product Manager in Brazil. You can find me at Medium and Twitter.
